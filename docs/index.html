HISTORIA

El desarrollo de PHP comenzó en 1994, cuando Rasmus Lerdorf escribió varios ficheros binarios Common Gateway Interface (CGI) en el lenguaje C, los cuales fueron concebidos para gestionar su página web personal. Extendió sus funcionalidades hasta el tratamiento de formularios web y la comunicación con una base de datos, y llamó a ésta implementación “Personal Home Page Tools” o “PHP-Tools”.

El 8 de Junio de 1995 se lanzó oficialmente la primera versión de PHP, conocida como la versión 1.0. Esta versión, aunque aún no era un lenguaje de programación totalmente independiente, ya contenía varias características que vemos en PHP hoy en día: variables tipo Perl (de ámbito dinámico y estático), tratamiento de formularios web e incrustación del código en un documento HTML.

El proyecto, aunque empezó desarrollándolo Lerdorf a título personal, fué revisado en 1997 por Andi Gutmans y Zeev Suraski, quienes cambiaron el analizador sintáctico (parser) y terminaron por definir PHP como un lenguaje de programación independiente, lanzando la versión 3.0 . A partir de entonces, las siglas PHP pasaron a significar “Preprocesador de hipertexto”.

El proyecto crecía en popularidad con el paso del tiempo. En 1998 Gutmans y Suraski cambiaron el núcleo de PHP y crearon el Zend Engine (Motor Zend). Éste motor mejoraba la ejecución de aplicaciones complejas y aportaba modularidad al código fuente. En 2004, se añadieron funcionalidades del paradigma orientado a objetos, como un modelo de datos orientado a objetos.

Las versiones se fueron sucediendo, convirtiendo PHP en una tecnología cada vez más robusta y potente. Actualmente, la versión más reciente de PHP es la 7.2 . Sus características se verán en profundidad en el apartado “Características de PHP”.

CARACTERÍSTICAS

PHP es un lenguaje de script del lado del servidor, usado para desarrollar páginas web estáticas, dinámicas o aplicaciones web. Los scripts en PHP sólo se pueden interpretar en un servidor que tenga PHP. Las máquinas cliente sólo pueden acceder al fichero PHP a través de un navegador. Un fichero en PHP contiene etiquetas PHP y termina con la extensión “.php”.

¿Qué es un lenguaje de script?

Un script es un conjunto de instrucciones que son interpretadas en tiempo de ejecución. Un lenguaje de script es un lenguaje que interpreta scripts en tiempo de ejecución. Los scripts suelen estar embebidos en otros entornos de software. El objetivo de los scripts suele ser mejorar el rendimiento o realizar tareas rutinarias para una aplicación.
